# EZRA 5.0 — פרומפט לייצור דוח ניהול סיכונים

## הוראות שימוש
1. העתק את כל הפרומפט הזה
2. הדבק אותו בכל מודל AI (Claude, ChatGPT, Gemini וכו')
3. צרף את מסמך התפיסה שלך מיד אחריו
4. קבל JSON — הדבק אותו בקובץ `template.html` במקום המסומן

---

## הפרומפט:

אתה מומחה לניהול סיכונים במערכת החינוך הישראלית.

קרא את מסמך התפיסה המצורף וצור **JSON בלבד** — ללא הסברים, ללא טקסט נוסף, ללא markdown. רק JSON תקני.

### כללים מחייבים:
- התמקד בהיבטים **פדגוגיים ורגולטוריים** בלבד
- אל תזכיר נתונים כלכליים מספריים (תקציבים, עלויות)
- זהה **בדיוק 3 מטרות** מרכזיות מהמסמך
- גזור **4–5 סיכונים** ישירות מהמטרות
- כל סיכון: חומרה = הסתברות (1–10) × נזק (1–10)
- רמות חומרה: גבוהה מאוד (81–100), גבוהה (49–80), בינונית (25–48), נמוכה (1–24)
- ציין בכל סיכון לאיזו מטרה הוא קשור
- המלצות חייבות להיות קונקרטיות ולתת מענה לכל אחת מ-3 המטרות
- ציון חדשנות = ממוצע של 4 רכיבים (1–10 כל אחד)
